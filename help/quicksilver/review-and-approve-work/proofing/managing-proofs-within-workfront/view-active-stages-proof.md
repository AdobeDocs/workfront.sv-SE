# Visa aktiva faser i ett korrektur

Du kan visa de aktuella aktiva stegen på ett korrektur som innehåller ett automatiserat arbetsflöde.

## Åtkomstkrav

+++ Expandera om du vill visa åtkomstkrav för funktionerna i den här artikeln.

Du måste ha följande åtkomst för att kunna utföra stegen i den här artikeln:

## Visa aktiva faser i ett korrektur

1. Håll muspekaren över dokumentet som innehåller det korrektur som du vill visa och välj sedan **Korrektur** för att visa korrekturfaserna.
