## Interagera med rapporter

Du kan interagera med enskilda rapporter på kontrollpanelen.

### Anpassa tabellrapporter tillfälligt

Du kan tillfälligt anpassa tabellrapporter på kontrollpaneler. De här ändringarna gäller endast för den aktuella sessionen och påverkar inte den ursprungliga rapportkonfigurationen.

1. Klicka på **Arbetsytans kontrollpaneler** i den vänstra panelen.
1. Klicka på namnet på en befintlig kontrollpanel för att öppna den.
   ![Öppna en instrumentpanel](assets/open-dashboard.png)
1. Leta reda på tabellrapporten som du vill anpassa.
1. Välj något av följande alternativ för att anpassa rapporten:

   | Alternativ | Beskrivning |
   |--------|-------------|
   | **Lägg till kolumn** | Klicka på **Lägg till kolumn** om du vill lägga till fler kolumner i rapporten. |
   | **Konfigurera kolumner** | Välj om du vill visa eller dölja specifika kolumner i rapporten. |
   | **Radhöjd** | Justera radhöjden för rapporten. |
   | **Fästrullning** | Aktivera eller inaktivera bläddring för enklare navigering i rapporten. |

   >[!IMPORTANT]
   >
   >De här ändringarna gäller endast för den aktuella sessionen och påverkar inte den ursprungliga rapportkonfigurationen. Om du vill göra dessa ändringar permanenta måste du redigera rapporten.

## View reports with grouped data

Report creators can configure reports to display grouped data. When viewing these reports on a Canvas Dashboard, you can expand or collapse the grouped data to see more or less detail.

Data within groups is sorted alphabetically or chronologically by default, depending on the data type. You can click the column headers to sort the data within each group based on different attributes. When you sort by a different attribute, the order of the groups remains unchanged.

When you sort by the same field that your report is grouped by, the group order can flip. For example, a text-based grouping that normally runs A–Z can switch to Z–A. This only happens when the sort column and the grouping attribute are the same.

## Visa rapporter med grupperade data

Rapportskapare kan konfigurera rapporter så att grupperade data visas. När en rapport har en gruppering kan du expandera eller komprimera grupperade data för att se fler eller färre detaljer.

Som standard sorteras själva grupperingarna alfabetiskt eller kronologiskt, beroende på fälttyp. Data inom varje gruppering sorteras oberoende av grupperingssorteringsordningen.

Du kan klicka på en kolumnrubrik om du vill sortera data i varje grupp. När du sorterar efter ett fält som skiljer sig från grupperingsfältet ändras inte grupperingsordningen.

När du sorterar efter samma fält som rapporten grupperas efter kan gruppordningen dock ändras. En textbaserad gruppering som normalt kör A-Z kan till exempel växla till Z-A.
