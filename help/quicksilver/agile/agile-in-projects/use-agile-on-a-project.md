# Använd Agile i ett projekt

Du kan utnyttja Agile-funktionaliteten i ditt projekt utan de administrativa problem som vanligtvis följer med Agile-rutiner (som att hantera en eftersläpning i teamet, skapa iterationer osv.). Mer information finns i [Hantera ett projekt i [!UICONTROL Agile] vyn ](../../manage-work/projects/manage-projects/manage-projects-in-agile-view.md).
