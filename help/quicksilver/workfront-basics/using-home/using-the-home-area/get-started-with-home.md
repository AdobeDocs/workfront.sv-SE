# Kom igång med [!UICONTROL Home]

<!--Audited: 12/2023-->

I den här artikeln beskrivs en översikt över det aktuella [!UICONTROL Home]-området i [!DNL Adobe Workfront]. Mer information om hur du arbetar i nya [!UICONTROL Home] finns i [Kom igång med nya startsidor](../new-home/get-started-with-new-home.md).

Det finns två sätt att visa ditt arbete i det aktuella [!UICONTROL Home] området:

* Från [!UICONTROL Work List]
* Från [!UICONTROL Calendar].

## [!UICONTROL Work List]

Du kan visa allt ditt tilldelade arbete på en plats i [!UICONTROL Work List]:

![[!UICONTROL Work List] paneler](assets/worklist-and-right-panel-home.png)

Granska följande avsnitt i det aktuella [!UICONTROL Home]-området, som markerats i bilden ovan:

* **A**: Använd den vänstra panelen för att filtrera, sortera och välja arbete som du har tilldelats.

* **B**: När du har valt en arbetspost i den vänstra panelen kan du använda den högra panelen för att interagera med anpassade fält, göra uppdateringar och logga tid.

Mer information om hur du använder [!UICONTROL Home Work List] finns i följande artiklar:

* [Visa objekt i [!UICONTROL Work List] i området [!UICONTROL Home]](../../../workfront-basics/using-home/using-the-home-area/display-items-in-home-work-list.md)
* [Skapa arbetsobjekt från området [!UICONTROL Home]](../../../workfront-basics/using-home/using-the-home-area/create-work-items-in-home.md)

## [!UICONTROL Calendar]

Du kan visualisera ditt arbete genom att visa när arbetsobjekten förfaller och du kan blockera tid för att slutföra dem genom att granska ditt arbete i [!UICONTROL Calendar] i [!UICONTROL Home]-området:

![](assets/calendar-home-nwe-350x114.png)

Granska följande avsnitt i det aktuella [!UICONTROL Home]-området, som markerats i bilden ovan:

* **A**: Använd vyn [!UICONTROL Calendar] för att skapa en visuell karta över det arbete du behöver utföra. Klicka och dra arbetsobjekten från [!UICONTROL Work List] till [!UICONTROL Calendar] för att blockera den tid det tar att arbeta.

* **B**: Använd synkroniseringsalternativet för [!UICONTROL Calendar] för att synkronisera med din [!DNL Outlook]-kalender. Hantera möten och arbeta på ett och samma ställe.

* **C**: Använd knappen [!UICONTROL Details] för att öppna en ny högerpanel där du kan visa mer information om arbetsobjektet.

* **D**: Använd förfallofältet för att hålla reda på när arbete som du har tilldelats förfaller.

Mer information om hur du använder hemkalendern finns i [Använd [!UICONTROL Home Calendar]-vyn](../../../workfront-basics/using-home/using-the-home-area/use-home-calendar-view.md).

## Hitta ditt arbete

Området [!UICONTROL Home] är en butik där du kan hitta uppgifter, utgåvor och godkännanden som du har tilldelats. Filtrerings-, grupperings- och sorteringsalternativen fungerar tillsammans så att du kan ordna dina arbetsobjekt som du vill. Du kan använda grupperingar för att gruppera objekt, filter för att fokusera på arbete som behöver slutföras och slutligen sortera objekten i den stigande eller fallande ordningen i grupperingen.

>[!NOTE]
>
>Filtrerings- och sorteringsalternativen lagras i webbläsaren. Om du konsekvent använder samma webbläsare på samma dator (och inte rensar platsdata) ändras inte filtren och sorteringen, men om du byter webbläsare eller dator ändras filtren och sorteringen.

### Gruppera liknande arbetsuppgifter tillsammans med grupperingar

Du kan använda grupperingar för att visa liknande objekt under en gemensam rubrik i arbetslistan. Du kan gruppera objekt efter:

* [!UICONTROL Planned Completion]
* [!UICONTROL Planned Start]
* [!UICONTROL Commit Date]
* [!UICONTROL Project]
* [!UICONTROL My Priority]

Mer information om hur du använder grupperingar finns i [Visa objekt i [!UICONTROL Work List] i [!UICONTROL Home]-området](../../../workfront-basics/using-home/using-the-home-area/display-items-in-home-work-list.md).

![](assets/group-by-drop-down-expanded-in-home-with-planned-start-date-nwe-350x273.png)

### Begränsa fokus med filter

Med filtret [!UICONTROL Work List] kan du begränsa ditt fokus till specifika arbetsobjekt.

Nedan följer exempel på hur du kan filtrera objekt som du visar i Hem:

* Filtrera efter objekttyp och tillstånd.

  Om du till exempel vill visa alla uppgifter kan du välja filtret [!UICONTROL Tasks]. Om du vill vara mer specifik än den här och bara visa uppgifter som du kan börja arbeta med, väljer du filtren [!UICONTROL Ready to Start] under filtret [!UICONTROL Tasks].

* Filtrera endast efter objekttyp.

  Du kan till exempel välja [!UICONTROL Issues] om du vill se alla utgåvor i alla lägen ([!UICONTROL Working On] eller [!UICONTROL Requested]), eller [!UICONTROL Approvals] om du vill visa alla arbetsobjekt, åtkomstbegäranden, tidrapporter, dokument och korrekturgodkännanden.

Mer information om hur du använder filter finns i [Visa objekt i [!UICONTROL Work List] i området [!UICONTROL Home]](../../../workfront-basics/using-home/using-the-home-area/display-items-in-home-work-list.md).

![](assets/displaying-work-items-filters-nwe-350x401.png)

### Få tillgång till arbete som tilldelats ditt team

Området [!UICONTROL Home] har en permanent gruppering ([!UICONTROL Team Requests]) som visar uppgifter och ärenden som tilldelats dina team. Team visas i det här avsnittet. När du klickar på ett teamnamn öppnas sidan Teamförfrågningar för det teamet.

>[!NOTE]
>
>Filtrerings- och sorteringsalternativen påverkar inte grupperingen [!UICONTROL Team Requests]. Den här grupperingen visas så länge du har arbete tilldelat ditt team.

Mer information om hur du får åtkomst till teamförfrågningar finns i artikeln [[!UICONTROL Manage] arbets- och teamförfrågningar i [!UICONTROL Home] området ](../../../workfront-basics/using-home/using-the-home-area/manage-work-and-team-requests-home.md).

![](assets/team-requests-expanded-home-group-by-drop-down-nwe-350x314.png)

### Övervaka material som du har skickat in

Håll reda på det arbete du har skickat in för godkännande direkt från [!UICONTROL Work List]. Det finns en permanent gruppering för [!UICONTROL Approvals I've Submitted] när du väljer att visa [!UICONTROL Approvals] i [!UICONTROL Work List]. Du kan påminna godkännaren om arbetet som behöver godkännas. Du kan även återkalla godkännandet om det behövs.

>[!NOTE]
>
>Filtret och sorteringen påverkar inte grupperingen [!UICONTROL Approvals I've Submitted]. Den här grupperingen visas så länge du väntar på godkännande.
>
>Antalet godkännanden i filtret [!UICONTROL Approvals] visar inte antalet objekt som du har skickat in för godkännande. Filtret [!UICONTROL Approvals] visar bara de objekt som väntar på ditt godkännande.

![](assets/approvals-expanded-home-group-by-drop-down-nwe-350x415.png)

## Prioritera det som är viktigt för dig

Med [!UICONTROL Work List] kan du visa och prioritera arbete som är viktigt för dig med grupperingen [!UICONTROL My Priority]. Du kan lägga till upp till 20 objekt i listan [!UICONTROL My Priority].

>[!NOTE]
>
>Alla arbetsobjekt som läggs till i [!UICONTROL My Priority] är bara synliga för dig och ingen kan prioritera arbete åt dig.

Mer information om hur du använder [!UICONTROL My Priority] finns i [Prioritera arbete i [!UICONTROL Home]-området](../../../workfront-basics/using-home/using-the-home-area/prioritize-work-in-home.md).

![](assets/prioritizing-in-home-group-by-drop-down-nwe-350x412.png)

## Få det gjort

När du har klickat på namnet på ett objekt i [!UICONTROL Work List] använder du den högra panelen i [!UICONTROL Home] för att uppdatera och slutföra arbetet.

### Arbeta i ett utrymme som är anpassat för dig

Din [!DNL Workfront]-administratör kan lägga till upp till 16 anpassade fält i objekt i [!UICONTROL Work List]så att det blir enklare att hitta och uppdatera den information du behöver. De flesta fälten kan redigeras, så du behöver inte navigera till arbetsposten.

Om du vill uppdatera något av fälten i det här området klickar du bara på ett fält. Om fältet är redigerbart kan du ändra det på den högra panelen.

>[!NOTE]
>
>Du kan inte uppdatera [!UICONTROL Commit Date] om det här fältet läggs till i ett arbetsobjekt från en layoutmall.

![](assets/16-fields-home-350x137.png)

### Uppdatera andra med viktig information

Kommunicera snabbt och effektivt med andra från [!UICONTROL Home]-området. Du kan ställa frågor, meddela andra om de senaste uppdateringarna eller svara på direkta kommentarer från [!UICONTROL Home]-området med hjälp av kommentarområdet för en arbetsuppgift. Dessutom kan du uppdatera andra om de framsteg du gör genom att:

* Uppdaterar [!UICONTROL Commit Date] i kommentarsområdet

  >[!NOTE]
  >
  >   Det här är den enda platsen där du kan uppdatera implementeringsdatumet i hemmet.

* Rapportering om hur du känner att saker går genom att lägga till en kommentar eller uppdatera [!UICONTROL Condition] för ett objekt
* Uppdaterar fältet Procent färdigt

Mer information om hur du kommunicerar i området [!UICONTROL Home] finns i [Uppdatera arbete](../../../workfront-basics/updating-work-items-and-viewing-updates/update-work.md).

![](assets/updates-home-350x198.png)

### Logga din tid

Håll koll på de timmar du har arbetat med enskilda arbetsuppgifter. Du kan välja en allmän timtyp, till exempel [!UICONTROL Task Time], eller så kan du välja en projektspecifik timtyp som har konfigurerats av [!DNL Workfront]-administratören.

Mer information om hur du loggar tid i området [!UICONTROL Home] finns i avsnittet [Hem](../../../timesheets/create-and-manage-timesheets/log-time.md#home) i artikeln [Loggtid](../../../timesheets/create-and-manage-timesheets/log-time.md).

![](assets/log-time-home-350x181.png)
